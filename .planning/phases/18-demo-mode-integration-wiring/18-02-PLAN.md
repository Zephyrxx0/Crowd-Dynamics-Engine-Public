     - `useEffect`:
       - **Subscription 1: Phase change detection** (existing approach — watches matchSlice.phase):
         ```ts
         const unsubPhase = liveStore.subscribe(
           (state) => state.match?.phase ?? null,
           (phase) => {
             if (!previousPhaseRef.current) {
               previousPhaseRef.current = phase;
               return;
             }

             if (phase && previousPhaseRef.current && phase !== previousPhaseRef.current) {
               // Phase changed — if using demo event with zoneDeltas, skip (demo handles it directly)
               if (currentDemoEvent?.zoneDeltas) {
                 previousPhaseRef.current = phase;
                 return;
               }

               // Map match phase to event type
               let eventType: string;
               if (phase === "half-time") eventType = "halftime";
               else if (phase === "full-time") eventType = "full-time";
               else if (previousPhaseRef.current === "first-half" && phase === "second-half") eventType = "second-half-start";
               else { previousPhaseRef.current = phase; return; } // No density change for this transition

               const state = liveStore.getState();
               const baseInput = state.v1ZoneData ?? presets.normal;
               const adjusted = applyPhaseTransitionDeltas(baseInput, eventType);
               state.initializeSim(adjusted);

               previousPhaseRef.current = phase;
             }
           }
         );
         ```

       - **Subscription 2: Score change detection** (NEW — handles goals within same phase for LIVE-02):
         ```ts
         const unsubScore = liveStore.subscribe(
           (state) => state.match?.score ?? null,
           (score) => {
             if (!previousScoreRef.current) {
               previousScoreRef.current = score;
               return;
             }

             if (score && previousScoreRef.current && score !== previousScoreRef.current) {
               // Score changed (goal scored) — if in demo mode with event zoneDeltas, skip (useDemoSequence handles it)
               if (currentDemoEvent?.zoneDeltas) {
                 previousScoreRef.current = score;
                 return;
               }

               // In live mode — apply generic +20% to all zones for goals
               const state = liveStore.getState();
               const baseInput = state.v1ZoneData ?? presets.normal;
               // Apply generic goal deltas (target all zones +20%)
               const adjusted = applyPhaseTransitionDeltas(baseInput, "goal", [
                 { zoneId: "north", deltaPercent: 20 },
                 { zoneId: "south", deltaPercent: 20 },
                 { zoneId: "east", deltaPercent: 20 },
               ]);
               state.initializeSim(adjusted);

               previousScoreRef.current = score;
             }
           }
         );
         ```

       - Cleanup: return `() => { unsubPhase(); unsubScore(); }` from useEffect

   - Key design decisions (the agent's discretion):
     - Phase subscription fires for: half-time, full-time, and second-half-start transitions
     - Score subscription fires when score string changes (any goal, in either mode)
     - Both subscriptions check `currentDemoEvent?.zoneDeltas` to avoid double-applying in demo mode
     - In live mode, score changes apply generic +20% to all zones (no zone-specific targeting since that data isn't available from the live match API)
