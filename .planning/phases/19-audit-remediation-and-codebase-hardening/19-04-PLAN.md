   - The status "dot" (a colored circle div or span) should include sr-only text:
     ```tsx
     // Replace the plain color dot with:
     <span
       className={`inline-block h-2 w-2 rounded-full mr-2 ${

     ```
   - Then the visible text badge can remain color-based while the sr-only class ensures screen readers get the status

5. **Add reduced-motion CSS to `src/app/globals.css`**:

   - Add at the end of the file (or after any existing @media blocks):
